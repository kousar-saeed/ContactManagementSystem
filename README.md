
# Contact Management System (Skeleton)

This repository contains a **project skeleton** for a Contact Management System:

- `backend/`: Java **Spring Boot** (Maven) API skeleton
- `frontend/`: **React** (Vite) UI skeleton

No business logic has been implemented yet—this is structure + dependencies only.

## Backend stack

- Spring Web
- Spring Security
- Spring Data JPA
- MS SQL Server JDBC Driver
- Lombok
- Bean Validation

## Frontend stack

- React
- `react-router-dom`
- `axios`
- `react-hook-form`
